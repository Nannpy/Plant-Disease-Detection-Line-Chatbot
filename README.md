# Flask KitchenSink

Sample bot using [Flask](http://flask.pocoo.org/)

## Getting started

Install required packages.
```
$ pip install -r requirements.txt
```

Create .env file.
```
LINE_CHANNEL_SECRET='<<Your Line Channel Secret>>'
LINE_CHANNEL_ACCESS_TOKEN='<<Your Line Channel Access Token>>'
```

Start the api.
```
$ python app.py
```
